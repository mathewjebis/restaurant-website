Spicy Hut Restaurant
[Live Website](https://mathewjebis.github.io/restaurant-website/)

A beautiful and responsive website for a restaurant that showcases a delicious digital food menu, store locations, and full contact details. It features a sticky navigation header, animated scroll transitions, and a clean mobile friendly navigation menu.

Great Features

Full Digital Menu Card: Organizes your food options into clear categories like Starters, Main Course, and Chaat with matching prices and bold best seller labels.

Dynamic Sticky Header: The navigation bar stays pinned to the top of the screen as you scroll down and drops a subtle shadow to remain readable over images.

Fading Scroll Animations: Food menu cards and location boxes fade and slide smoothly into place automatically as you scroll down the page.

Mobile Hamburger Menu: Converts the wide navigation link bar into a clean drop down menu button automatically on small cell phone screens.

Quick Action Phone Anchors: Users can tap physical phone buttons to instantly call separate restaurant branch kitchens or landlines directly.

Smooth Page Scrolling: Clicking navigation section buttons slides the viewport down the page gracefully instead of cutting or jumping instantly.

How the Files Work

index.html maps out the layout structures, lists food items with prices, and handles the footer social media link buttons.

index.css manages the dark backdrop colors, beautiful hero image background blur filters, grid item placement, and responsive phone breakpoints.

index.js controls the mobile drop down toggle loops, cleans up open menus on link taps, and sets up scroll intersection observers.

How to Run the Project

1. Copy the project files to your computer.
2. Double click the index.html file to launch the restaurant homepage instantly inside your favorite web browser.
